# AFO Sandbox Builder MCP

Disposable sandbox build workcells for AFO, GitHub, Cloudflare, and Message OS.

This MPP app creates fast reversible sandbox plans for building Cloudflare Worker apps from inside ChatGPT or Claude.

Live worker (after deploy):
https://afo-sandbox-builder-mcp.jaredtechfit.workers.dev/

MPP endpoint:
https://afo-sandbox-builder-mcp.jaredtechfit.workers.dev/mcp

Tools:
- create_sandbox_app
- list_sandboxes
- check_sandbox_status
- mark_sandbox_candidate
- clone_sandbox_plan
- delete_sandbox_plan
